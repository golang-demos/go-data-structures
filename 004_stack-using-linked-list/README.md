## Stack - Using Linked list

This code implements Stack using Linked list for storing data.


### Operations
- Push
- Pop
- Print


### To run 
```
cd 004_stack-using-linked-list
go run main.go
```

### Output
```
Pushing 12 to stack
  Stack : 12 | 
Popped Value :  12
  Stack : <empty>
Pop() Called : No item to retrieve
  Stack : <empty>
Pushing 37 to stack
  Stack : 37 | 
Pushing 82 to stack
  Stack : 37 | 82 | 
Popped Value :  82
  Stack : 37 | 
Pushing 29 to stack
  Stack : 37 | 29 |
```
