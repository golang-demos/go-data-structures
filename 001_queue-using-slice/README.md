## Queue - Using Slice

This code implements Queue using slice for storing data.


### Operations
- Enqueue
- Dequeue
- Print


### To run 
```
cd 001_queue-using-slice
go run main.go
```

### Output
```

Adding  78  to queue

Adding  42  to queue
```
